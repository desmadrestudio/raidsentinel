How it works

Raid Sentinel is built for clear setup and transparent fulfillment.

1

Select a screen region

Choose the area of your game window that contains raid, combat, or server log text.

2

Run local OCR

The app captures screenshots on a set interval and reads visible text on your PC.

3

Send Discord alerts

Matching lines are posted to your configured Discord webhook with duplicate suppression.

Screen capture only

Raid Sentinel only reads pixels from the selected screen region. It does not inject into games, modify game files, inspect game memory, bypass anti-cheat systems, or automate gameplay.
